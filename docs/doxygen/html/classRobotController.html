Detailed Description

A simple controller which listens for the response output generated by InteractionCore and passes it to a RobotInterface as a reference signal for control. The response from InteractionCore is a trajectory of states which the robot should follow. This controller spins at the frequency specified by control_frequency and at each iteration sends the next state in the trajectory to the RobotInterface. If the RobotInterface is within a given threshold of the current robot state (see RobotInterface for details), then the next state in the trajectory is incremented by 1. If the threshold in RobotInterface is 0, then this is equivalent to sending each state in the response trajectory at control_frequency.
